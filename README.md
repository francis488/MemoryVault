# MemoryVault Pro 🔒

A high-performance, private, and fully client-side web application for managing personal memories, photos, videos, and audio tracks.

## 🚀 Features
- **Centric Design:** Clean, centered user interface for all authentication and dashboard views.
- **Privacy First:** No servers. Your data stays in your browser's IndexedDB.
- **Multi-Media Support:** Upload and preview Photos, Videos, and Audio.
- **Smart Gallery:** Shows a preview of 2 items with a "+X" overlay for larger albums.
- **Backup & Restore:** Export your entire vault to a `.json` file and restore it on any device.
- **Storage Monitoring:** Live tracking of browser storage usage.

## 🛠️ Technology Stack
- HTML5, CSS3, JavaScript (ES6)
- **Database:** IndexedDB (for media) & LocalStorage (for sessions)
- **Styling:** Flexbox, Grid, and Glassmorphism effects.

## 📂 File Structure
- `gateway.html`: Landing page.
- `signup.html`: User registration.
- `login.html`: Secure access.
- `reset.html`: Password recovery.
- `index.html`: Main dashboard & media manager.
- `script.js`: Database logic & file processing.
- `style.css`: Global styles and responsiveness.

## 📥 Installation
1. Download all files into a single folder.
2. Open `gateway.html` in a modern browser (Chrome/Edge/Safari).
3. Start saving your memories!
